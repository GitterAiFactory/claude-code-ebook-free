Prologue

AI 때문에
내 자리가 없어지면
어떡하나 싶었습니다

솔직하게 말하겠습니다. 저는 AI를 가장 강하게 거부했던 사람이었습니다. AI 얘기만 나오면 자리를 피하고, "이건 진짜 디자인이 아니야"라고 단정 짓던 사람이었습니다.

이유가 있었습니다. 재능 플랫폼 작업 736개, 평점 4.9. KBS·서울시·넥슨·롯데자이언츠 등 디자인 작업을 해온 10년. 월 천도 찍어본 프리랜서였습니다.

그런데 어느 날부터 클라이언트들의 말이 달라졌습니다. "AI로 빠르게 뽑을 수 없어요?" 처음엔 흘려들었습니다. 그러다 문득 이런 생각이 들었습니다. "이러다 내 10년 가치가 깎이는 건 아닐까." 불안감은 그렇게 시작됐습니다.

써봐야겠다고 마음먹으면서도, AI 툴 앞에서 손가락이 멈췄습니다. 몇 달 동안이나. "이걸 쓰는 순간 내 10년이 의미없어지는 건 아닐까." 그 두려움을 인정하는 데만도 시간이 걸렸습니다.

Prologue — 계속

그러다 스레드 피드에서 한 게시물이 눈에 들어왔습니다. AI 자동화로 수익을 만들고 있다는 글이었습니다. 댓글에 "저도 해봤어요", "나도 시작했어요"가 달리고 있었습니다. "이미 하는 사람"과 "아직 안 한 사람"의 격차가 눈에 보이는 순간이었습니다.

그날 결심했습니다. Claude Code를 처음 열던 날, 손이 약간 떨렸습니다. 두려움을 안고 프롬프트를 입력했습니다.

솔직히 말하면 중간에 3번은 포기할 뻔했습니다. 에러가 쏟아졌고, 결과물이 마음에 들지 않았고, "내가 왜 이러고 있나" 싶은 순간이 있었습니다. 그래도 계속했습니다.

그렇게 쓰다 보니 감이 잡혔고, 전자책을 만들기로 결심한 뒤 4일 만에 1권이 완성되어 있었습니다. 출시 2일 만에 300만 원이 넘는 수익이 생겼습니다. AI를 가장 강하게 거부했던 제가, 지금은 그 AI로 수익을 만드는 사람이 되었습니다.

매일 안정적이진 않지만, 이 방향이 맞는다는 확신은 매일 강해집니다. AI가 두려웠던 분께, 이 이야기가 작은 용기가 되길 바랍니다.

이 책이 당신의 첫 판매로 이어지길 바랍니다

0.1

클로드 · 요금제 ·
팔리는 주제 고르기

이 책은 Claude Code로 PDF 전자책을 직접 만들고, 판매 수익화까지 실현한 제 실전 경험을 그대로 담은 책입니다. 이론이 아닌, 실제로 돈이 들어온 과정과 판매 노하우까지 공개합니다.

Claude는 Anthropic이 만든 AI입니다. Claude Code는 그 Claude를 터미널에서 직접 실행해 파일을 직접 만들고 수정하는 도구입니다. 프롬프트 한 줄이 실제 HTML 파일이 됩니다.

요금제는 Claude Pro (월 $20)면 충분히 시작 가능합니다. 사용량이 많아지면 Claude Max (월 $100~200)로 업그레이드하면 됩니다. 이 책에서 다루는 전자책 제작은 Pro 플랜으로 충분합니다.

팔리는 전자책 주제에는 세 가지 공통점이 있습니다. 첫째, 독자가 이미 찾고 있는 답을 다룹니다. "부업"보다 "Claude Code로 4일 만에 전자책 만들기"가 팔립니다. 둘째, 저자가 직접 해봤다는 증거가 있습니다. 수익 스크린샷 하나가 이론 1000줄보다 강합니다.

셋째, 읽고 나서 즉시 따라할 수 있어야 합니다. 이론서가 아니라 매뉴얼이어야 합니다. 피해야 할 것: 범위가 너무 넓거나, 본인이 직접 해보지 않은 주제. 독자는 생각보다 빠르게 허구를 감지합니다.

Mac 설치 가이드

macOS 13 이상 · M1/M2/M3/M4/Intel 모두 · 약 5분 · 2026년 4월 최신
▲ 2026년 공식 설치법 — 명령어 1줄로 끝 (네이티브 설치)
Homebrew · Node.js 따로 설치할 필요 없습니다. 네이티브 바이너리가 직접 설치되며 자동 업데이트됩니다.

① 터미널 열기
Command ⌘ + Space → "Terminal" 검색 → 실행

② Claude Code 설치 — 한 줄로 완료 (공식 권장)

비밀번호 입력 시 화면에 안 보이는 게 정상. 자동으로 모든 의존성 설치. 자동 업데이트 포함.

▲ Homebrew 사용자라면
brew install --cask claude-code 로도 가능.
단, 자동 업데이트 안 됨 — brew upgrade claude-code 로 수동 업그레이드 필요.

③ 터미널 완전히 닫고 새로 열기

전체 흐름 —

STEP 1  ·  초안 파일 생성
지금 바로 따라하기 →
터미널에서 claude 실행 → 아래 프롬프트 입력 → 3~5분 후 초안 HTML 완성
복사해서 붙여넣기
지금부터 당신은 세계 최고의 편집디자인 총괄이자 베스트셀러 전자책 작가입니다. [주제]에 관한 전자책을 HTML 파일로 만들어주세요. 저장 경로: /Users/[이름]/Desktop/[파일명].html 타겟 독자: [대상 — 예: AI 완전 초보자] 분량: 30~40페이지 A4 디자인: 따뜻한 배경(#fff4f3), 강조색 주황(#FF5C35), 모서리 직각(border-radius:0) ⚠️ 페이지 크기는 반드시 A4(210mm×297mm)로 고정. 내용이 많아도 페이지가 세로로 늘어나면 안 됩니다. 넘치는 내용은 다음 페이지로 분리해주세요. 목차 먼저 제안하고, 승인 받은 뒤 시작해주세요.
[ ] 안만 본인 내용으로 바꾸면 됩니다. 나머지는 그대로 복붙.
⚠️ 브라우저에서 열었을 때 페이지가 세로로 길다면 → "A4로 고정해줘" 재요청
